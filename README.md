# Network File Sharing Server & Client with Authentication and Encryption

## Overview
A Python socket-based client-server file sharing system with:
- File listing, upload, and download
- Username/password authentication
- XOR-based encryption of file data

## How to run
1. Run server:
2. Run client:
3. Login (default):
- Username: `admin`
- Password: `1234`

## Commands
- `list` — show files on server
- `download <filename>` — download from server
- `upload <filepath>` — upload a file to server
- `exit` — close connection
